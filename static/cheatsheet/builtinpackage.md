## `java.lang`

`java.lang` is automatically included in every java file.

Contains:

* Fundamental Classes, `Object`, `Class`
* Wrapper Classes, `Boolean`, `Character`, `Byte`, `Short`, `Integer`, `Long`, `Float`, `Double`
* String Manipulation, `String`, `StringBuffer`, `StringBuilder`
* Error Handling, `Throwable`, `Exception`, `Error`, `RuntimeException`
* Math Operations, `Math` class functions

## `java.awt`

Contains Java's built in toolkit for creating GUI applications.

## `java.util`

Contains:

* Data Structures: `ArrayList`, `LinkedList`, `HashMap`, `HashSet`, `TreeMap`, `TreeSet`, `PriorityQueue`
* Utility CLasses: `Collections`, `Arrays`, `Objects`
* Date & Time: `Date`, `Calendar`, `TimeZone`
* Parsing: `StringTokenizer`, `Scanner`
* Random: `Randomness`
* Encoding: `Base64`

## `java.io`

Contains:

* Byte Steams: `InputStream`, `OutputStream`, `FileInputStream`
* Character Streams: `Reader`, `Writer`, `FileReader`
* File System Management: `File`
* Exceptions: `IOException`, `FileNotFoundException`
